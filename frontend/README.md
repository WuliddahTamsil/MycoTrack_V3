# MycoTrack Backend Server

Backend server untuk aplikasi MycoTrack yang berjalan di port 3000.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Jalankan server:
```bash
npm start
```

Untuk development dengan auto-reload:
```bash
npm run dev
```

## Struktur Data

- `data/admin.json` - Data admin
- `data/customers.json` - Data customer
- `data/petanis.json` - Data petani
- `uploads/` - Folder untuk menyimpan file upload (foto profil, KTP, dll)

## Endpoints

### Customer
- `POST /api/customer/register` - Registrasi customer
- `POST /api/customer/login` - Login customer

### Petani
- `POST /api/petani/register` - Registrasi petani
- `POST /api/petani/login` - Login petani

### Admin
- `POST /api/admin/login` - Login admin

### Health Check
- `GET /api/health` - Status server
